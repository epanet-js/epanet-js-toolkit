How to Compare

Open Chrome DevTools → Memory tab. Take a heap snapshot before and after each run. The pthreads + OPFS build should show significantly less WASM heap growth for large networks because output files are backed by the on-disk OPFS rather than RAM.

The Peak Main Heap stat uses performance.memory (Chrome only). For Firefox/Safari, rely on their native DevTools profilers.
